# JIANCAN
简餐--大三下学期实训项目
## 项目介绍
**简餐App**是一款旨在通过用户对美食制作教程的分享使用户学到方便、简单、心仪的食物制作方法，节省用户制作食物的时间。的APP。
<br>
<br>
用户可以在APP上分享、浏览各类菜式的制作方法。
